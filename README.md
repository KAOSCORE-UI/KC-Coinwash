# KC-Coinwash
🫧 KC-Coinwash — Money Laundering System
## Features

- **Admin Tablet** — place / remove washing machine props anywhere on the map, live config control
- **Third-Eye Interaction** — proximity E-key prompt on every placed prop (no framework target required)
- **Player Wash UI** — dirty/black money → clean cash or bank, with fee breakdown & risk levels
- **Animated Drum Overlay** — spinning washing machine HUD during the wash cycle
- **Persistent Locations** — washer positions saved to JSON, survive resource restarts
- **Framework Bridge** — works with ESX, QBCore, or custom; swap inventory with one config line
- **Server-side truth** — all economy logic runs server-side; no client exploitation
